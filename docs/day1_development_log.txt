Day 1 下午开发记录

1. 完成 MySQL 核心表初始化：
users、questions、exams、exam_questions、exam_records、student_answers、wrong_questions、operation_logs。

2. 完成演示账号：

3. 完成后端登录接口：
POST /api/auth/login
支持账号密码登录，返回 success、message、data 统一格式。

4. 完成登录日志：
登录成功记录 LOGIN_SUCCESS，登录失败记录 LOGIN_FAIL。

5. 完成日志查询接口：
GET /api/logs/latest
管理员后期可查看最近 50 条操作日志。

6. 完成 Qt 登录页：
使用美化后的左右分栏登录界面，包含登录和注册入口。

7. 完成 Qt 调用后端：
Qt 使用 QNetworkAccessManager 调用 Spring Boot 登录接口。

8. 完成三角色跳转：
teacher 进入教师工作台；
student 进入学生学习中心；
admin 进入管理员控制台。

9. 遇到的问题与解决：
Qt 项目使用 CMake，没有 .pro 文件，因此在 CMakeLists.txt 中加入 Network 组件。
浏览器不能直接测试 POST，因此使用 IDEA HTTP Client 测试接口。
